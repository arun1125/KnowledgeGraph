## Properties

| Properties      | Values |
| ----------- | ----------- |
| Parameters      | a,b       |
| Support   | x $\in$ [a,b]       |
| PDF   | $\frac{1}{b-a}$  |
| CDF   | $\frac{x-a}{b-a}$ or 1      |
| Mean   | $\frac{b+a}{2}$      |
| Variance   | $\frac{(b-a)^2}{12}$       |
| Median   | $\frac{b+a}{2}$       |
| Mode   | any value in [a,b]       |



## Use Cases
- used as a flat [[Prior]] when you have no prior knowledge what what you are modelling in Bayesian Statistics
